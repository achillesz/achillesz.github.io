1. node 成功安装之后，npm 可用
2. npm install -g grunt-cli
3. 进入 /bootstrap/ 根目录，然后执行 npm install 命令。npm 将读取 package.json 文件并自动安装此文件中列出的所有被依赖的扩展包。
4. npm config set registry https://registry.npm.taobao.org
5. npm install

type是个静态方法，主要用来实现类型判断

	type of obj
	
|obj|result1|
|:---:|:---:|
|undefined|undefined|
|null|object|
   

内引用 `的的`

{% highlight js %}
        type: function(obj) {
            if (obj == null) {
                return String(obj);
            }
            // Support: Safari <= 5.1 (functionish RegExp)
            return typeof obj === "object" || typeof obj === "function" ?
                class2type[core_toString.call(obj)] || "object" :
                typeof obj;
        }
{% endhighlight %}
